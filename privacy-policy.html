1. Introdução

Esta Política de Privacidade descreve como o Laboratório Ev.C.S ("nós", "nosso" ou "aplicativo") coleta, usa, armazena e protege suas informações pessoais quando você utiliza nosso aplicativo de cálculos de ensaios geotécnicos.

2. Dados Coletados

Coletamos os seguintes tipos de informações:

  • Dados de conta: endereço de e-mail para autenticação.
  • Dados de ensaios: informações sobre operadores, responsáveis técnicos, verificadores e dados de ensaios geotécnicos.
  • Dados de localização: coordenadas geográficas relacionadas aos ensaios.
  • Dados de uso: informações sobre como você utiliza o aplicativo.

3. Finalidade do Tratamento

Utilizamos seus dados para:

  • Fornecer e manter o serviço
  • Autenticar usuários e garantir segurança
  • Armazenar e processar dados de ensaios geotécnicos
  • Gerar relatórios e documentação técnica
  • Realizar backups e garantir a integridade dos dados

4. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

  • Provedores de serviços: utilizamos o Firebase (Google) para autenticação e armazenamento de dados.
  • Requisitos legais: podemos divulgar seus dados quando exigido por lei.

5. Transferência Internacional

Seus dados podem ser transferidos e mantidos em servidores localizados fora do seu país, onde as leis de proteção de dados podem ser diferentes. Ao utilizar nosso aplicativo, você concorda com essa transferência.

6. Segurança dos Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados, incluindo:

  • Criptografia de dados sensíveis
  • Autenticação segura
  • Backups regulares
  • Logs de auditoria

7. Seus Direitos

De acordo com a LGPD, você tem os seguintes direitos:

  • Acesso aos seus dados pessoais
  • Correção de dados incompletos ou incorretos
  • Exclusão de dados
  • Portabilidade de dados
  • Revogação do consentimento

Para exercer seus direitos, entre em contato conosco através do e-mail: [EMAIL]

8. Retenção de Dados

Mantemos seus dados pelo tempo necessário para cumprir as finalidades descritas nesta política, a menos que um período de retenção mais longo seja exigido por lei.

9. Alterações na Política

Podemos atualizar esta política periodicamente. Notificaremos sobre alterações significativas através do aplicativo ou por e-mail.
